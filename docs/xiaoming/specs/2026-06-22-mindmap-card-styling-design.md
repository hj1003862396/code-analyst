# 脑图卡片样式重构设计文档

本文档定义了将 code-analyst 调用链脑图节点卡片重构为全新规整样式的设计规范。

## 1. 目标与背景
为了提升 code-analyst 脑图可视化效果，需要将原有的毛玻璃渐变卡片样式进行重构，采用更规整、扁平且富含现代质感的新样式。新样式参考了具有清晰层级和属性胶囊（Badge）的全局变量卡片设计。

## 2. 设计规范

### 2.1 卡片结构 (HTML/DOM)
在 `index.js` 的 `createCustomNodeDom` 方法中重构卡片 DOM，使其包含以下层次：
1. **类型头部 (`.card-header-type`)**：
   - 包含一个左侧竖直彩色指示条（小色块）及文字标签。
   - 文字标签根据节点类型确定：
     - 根节点：`🎯 入口方法`
     - Mapper 节点：`💾 Mapper 接口`
     - Service/常规节点：`⚡ Service 方法`
2. **两列属性列表行 (`.card-row`)**：
   - **类名行**：左侧为固定文本 `类名`，右侧为带有灰色胶囊背景的类包名简称，鼠标 hover 时通过 `title` 属性显示完整类路径。
   - **方法名行**：左侧为固定文本 `方法名`，右侧为带有灰色胶囊背景的方法名，采用等宽字体。
3. **内容主体区 (`.card-body`)**（仅在有备注或 SQL 语句时渲染）：
   - 使用一条浅灰色细虚线作为分隔线。
   - **方法简介 (Remarks)**：简洁展示在备注区。
   - **SQL 语句 (SQL Operations)**：在深色高对比度毛玻璃容器内展示对应 SQL 标签及 SQL 代码。

### 2.2 样式表现 (CSS)
在 `index.css` 中重构卡片相关的 CSS：
* **主体背景与阴影**：
  - 卡片背景：纯白色 solid (`#ffffff`)，无透明毛玻璃。
  - 边框：`1px solid #e2e8f0`（浅灰）。
  - 圆角：`12px`。
  - 阴影：`0 4px 20px rgba(0, 0, 0, 0.05)`（柔和大阴影）。
  - 宽度限制：最小宽度 `220px`，最大宽度 `360px`。
* **左侧竖直指示条**：
  - 在卡片左侧利用 `border-left: 4px solid var(--accent-color)` 实现指示条。
  - 颜色变量定义：
    - 入口节点 (Root)：`#10b981` (Emerald Green)
    - Service 节点：`#3b82f6` (Blue)
    - Mapper 节点：`#f97316` (Orange)
* **属性对齐样式**：
  - `.card-row` 采用两端对齐：`display: flex; justify-content: space-between; align-items: center; gap: 12px; margin-top: 4px;`
  - `.row-label`：字号 `12px`，字重 `500`，颜色 `#64748b`。
  - `.row-badge`：字号 `12px`，字重 `500`，内边距 `3px 8px`，圆角 `6px`，背景色 `#f1f5f9`，文字颜色 `#334155`，字体 `var(--font-mono)`。右侧内容超长时自动显示省略号。
* **折叠按钮**：
  - `.card-expand-btn` 依然绝对定位，定位在卡片右侧边界，将原本相对于容器的 `right: 0` 修改为 `right: 12px` 以免超出外层 SVG `<foreignObject>` 产生被剪裁/半遮挡的问题。采用优雅的圆形白底小按钮，悬浮时稍微放大并凸显主色边框。

### 2.3 连线样式与背景激活框 (Connections & Active Border)
在 `index.js` 的 `MindMap` 实例化配置中：
- 连接线颜色 `lineColor` 调整为轻快温和的浅蓝紫色（如 `#818cf8`）。
- 连接线宽度 `lineWidth` 设置为 `2` 像素。
- 隐藏悬浮及选中时的外边框：配置 `hoverRectColor: 'transparent'`，彻底消去卡片在悬浮或选中时周围绘制的淡蓝色激活框。

---

## 3. 验证方案

### 3.1 自动/编译校验
- 检查修改后的 CSS 无语法错误。
- 确保浏览器端不发生 JS 语法及加载错误。

### 3.2 手动校验
1. 打开浏览器访问 `http://localhost:8080/index.html`。
2. 输入类名和方法名并初始化调用链，确认卡片呈现纯白色、带左侧指示条、且「类名」与「方法名」呈现为左右两端对齐的 Label + Badge 胶囊格式。
3. 点击 `+` 按钮展开子节点，验证折叠/展开按钮功能及子节点卡片排版。
4. 确认节点之间的连接线显示为浅蓝紫色曲线。
