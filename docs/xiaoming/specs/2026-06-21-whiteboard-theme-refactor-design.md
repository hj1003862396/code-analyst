# 2026-06-21-whiteboard-theme-refactor-design

## 1. 目标与背景
本项目的前端页面当前采用的是左右分栏的标准 Dashboard 布局，风格偏向传统的 Web 应用程序。
根据用户最新的需求和提供的参考截图，我们需要将界面整体改造为**沉浸式无限白板与悬浮工具栏/侧栏**的风格，打造干净、高级的思维导图式视觉体验：
1. **背景颜色**：从浅灰蓝色背景改为纯白色，去除繁杂网格。
2. **布局结构**：将左中右分栏布局升级为铺满全屏的白板画布，操作表单和详情区域均作为精致圆角带阴影的悬浮卡片悬挂于画布四周，支持折叠与收起。
3. **工具栏**：在页面四周浮动展示四个与截图高度一致的工具栏（包括退出/画板、分享/编辑/搜索、左侧垂直几何工具栏、右下角缩放/撤销重做工具栏）。
4. **思维导图分支连线**：修改直角 Elbow 折线的对齐逻辑，将水平折线直接作为子节点文字下方的“下划线”进行承载（普通方法为蓝色实线，Mapper 方法为红色虚线），使得树状图风格高度契合专业思维导图软件（如 XMind 风格）。

## 2. 详细设计方案

### 2.1 整体布局设计（HTML / CSS）
* **基础容器**：
  * 去除原有的 `.left-panel`、`.middle-panel`、`.right-panel` 传统的物理分栏布局，修改为重叠定位。
  * 整个页面主容器 `body` 采用 `#ffffff` 背景色。
  * `tree-svg` 画布层为 `position: absolute; left: 0; top: 0; width: 100%; height: 100%; z-index: 1;`，使绘图区域充满屏幕。

* **悬浮工具栏 (Floating Toolbars)**：
  在 SVG 画布之上浮动渲染四个精美的控制栏，圆角边框及阴影与截图保持高度一致：
  * **Top-Left (顶部左侧浮动控制区)**：
    * `X 退出` (带有简洁小叉号图标) 和 `画板` (带有绿色画笔微调高亮的高级设计风格)。
  * **Top-Right (顶部右侧浮动控制区)**：
    * 蓝色框体的 `🔗 分享` 药丸按钮、`📝 编辑 v`（下拉菜单风格，附带小铅笔图标）、搜索图标、评论图标、以及“...”更多选项按钮。
  * **Left-Side (左侧浮动垂直几何与绘图工具条)**：
    * 高级设计感的窄条垂直浮动框，从上到下集成以下图标占位符：几何图Logo、矩形、文本(T)、便签、折线连接、帧、表格、画笔、列表、应用格、聊天气泡。
  * **Bottom-Right (右下角状态与缩放栏)**：
    * `[ ↩️ 撤销 ] [ ↪️ 重做 ]`、`[ 🖐️ 抓手 ]`、`[ - 89% + ]`（缩放与 D3 zoom 状态实时联动绑定）、帮助 `?` 按钮。
    * 提供双向绑定：在 `d3.zoom` 回调中将缩放 scale 乘以 100 取整同步展示为百分比；点击 `+` / `-` 按钮时，手动触发 `d3.zoom.scaleBy` 达到缩放效果。

* **悬浮控制与分析面板（Collapsible Panels）**：
  * **左侧“分析入口表单” (`left-card`)**：
    * 定位为左侧浮动卡片（如 `position: absolute; left: 24px; bottom: 24px; width: 340px; z-index: 10;`）。
    * 提供“收起/折叠”动效。折叠时缩回左侧边缘，只显示一个小指示气泡，点击气泡可重新滑出展开。
  * **右侧“节点详细分析” (`right-drawer`)**：
    * 定位为右侧悬浮抽屉式卡片（如 `position: absolute; right: 24px; top: 80px; bottom: 80px; width: 440px; z-index: 10;`）。
    * 采用极细淡灰色边框，配以柔和的投影，完美贴合白板风格。同样提供折叠/展开按钮以释放视觉空间。

### 2.2 D3 节点与下划线折线渲染逻辑升级
* **节点定位与盒模型宽度**：
  * 在 D3 `.join()` update 阶段，计算文本实际边界宽度 `bbox.width`。
  * 根节点显示为实心皇家蓝 `#3b66e2` 圆角矩形卡片，内嵌白色文字。
  * 子方法节点无背景矩形。从文字起点左侧 10 像素处至文字终点右侧 10 像素处，绘制一条水平的 `line.underline`。
* **XMind 风格分支直角线条**：
  * 从父节点的右侧引出水平连接线，经历 90 度垂直折角后下落，然后以水平线直接对接子节点的 `line.underline` 下划线。
  * 代码公式实现：
    * `startY = d.source.y + d.source.width` （其中 `width` 为父节点卡片宽度，如果是普通节点则是下划线宽度）。
    * `endY = d.target.y`。
    * 普通节点连线使用**蓝色实线**（`stroke: #3b66e2`），指向数据库 Mapper 的节点连线使用**红色虚线**。

## 3. 验证方案
* **视觉验证**：在浏览器中访问并缩放画布，确认背景完全为白色，四个浮动工具栏排版比例、阴影以及高亮状态和用户截图一致。
* **交互验证**：
  * 点击“分析入口”卡片的收起按钮，确认卡片优雅消失；点击边缘气泡确认其重现。
  * 点击右下角 `+` 和 `-` 按钮，确认 SVG 画布对应放大和缩小，且百分比数值正确。
  * 展开/下钻普通方法和 Mapper 方法，确认直角肘线下划线正确渲染，无错位重合。
